# Create VK Mini App [![npm][npm]][npm-url] [![deps][deps]][deps-url]

## Начать работу

Введите в терминале

`npm install`

## Что использовано
* VKUI
* VK Icons
* [VK Mini Apps api](https://github.com/VKCOM/vk-mini-apps-api/ "VK Mini Apps api")
* axios
* [HappySanta router](https://github.com/HappySanta/router "Happy Santa Router")
* React Redux
